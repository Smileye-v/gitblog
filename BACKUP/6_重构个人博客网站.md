前端技术
框架：ts + react + nextjs（强烈建议，不要用其他）
表单：react-hook-form + yup
样式：tailwindcss + headlessui + css variable
浮动定位：floating-ui
异步状态管理：axios + react-query + zustand + immer
模拟：mswjs（可选）
部署：AWS Amplify（自动化，几乎免费）
